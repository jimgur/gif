vid2gif
=======

- node utility to generate a GIF from a youtube video via command line
- Stitch these frames together to create a GIF, which is then uploaded to Imgur
- Jim @ imgur

## Compile and run the node server
    
    $ scripts/run
    

## Running the unit tests

    $ scripts/test
